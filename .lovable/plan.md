

Vou trocar o favicon para um ícone de cesta de compras. Como você se refere à cesta do canto superior esquerdo (que é o ícone `ShoppingBasket` do lucide-react), vou criar um favicon SVG baseado nesse mesmo ícone.

Passos:
1. Criar um arquivo SVG de cesta de compras em `public/favicon.svg`
2. Atualizar o `index.html` para referenciar o novo favicon
